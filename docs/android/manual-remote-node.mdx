# Manual Remote Node Setup

Essentially what you need is a working LND node.
Please refer to the [LND](https://github.com/lightningnetwork/lnd) project for setup instructions.

How to connect Zap:<br />
Use the [lndconnect](https://github.com/LN-Zap/lndconnect) project to create the connect QR-Code/string
